So...What's Robo Brain?


Robo Brain is a large-scale computational system that learns from publicly available Internet resources, computer simulations, and real-life robot trials. It accumulates everything robotics into a comprehensive and interconnected knowledge base. Applications include prototyping for robotics research, household robots, and Google cars.

We hope you are just as excited about this project as we are. For more information, please view the Research page.
