RaQuel API Documentation

Currently, RaQuel supports the following functions:

1. FETCH

The fetch function is used to retrieve a sub-graph of RoboBrain that matches an input pattern. The function fetch takes as input a pattern and returns a list of tuples with instantiated values of variables. This pattern is matched against sub-graphs of RoboBrain and these matched sub-graphs are used to create initialized entries for the pattern variables.
