RaQuel API Documentation

Currently, RaQuel supports the following functions:

1. fetch function

fetch: Pattern(arg1,arg2,...) returns a list of tuples with instantiated values of variables [(arg1(i),arg1(i),...)]i. This pattern is matched against the RoboBrain graph and these matched sub-graphs are used to create initialized entries for the pattern variables.

For example, to retrieve all nodes and edges with label 'HAS_MATERIAL' connected to nodes with handle "wall":

>>> fetch("({handle:'wall'})-[e:`HAS_MATERIAL`]->(b)")

The pattern will match against paths of length one {u, e, v}; where node u has handle "wall" and edge e has label 'HAS_MATERIAL'.

2. sort function

SortBy(Pattern,'key') returns a list of tuples sorted according to the key.

For example, to sort all paths of length between 1 and 3, which connects nodes with handle 'sitting_human' to nodes with handle 'cup', according to belief:

>>> path = fetch("({handle:'sitting_human'})-[e*1..3]->({handle:'cup'})")
>>> SortBy(paths,'Belief')

3. len function

The len function is used to find the length of the list of nodes or edges.

4. map function

The map function calls a function on a specific set of values.

5. filter function

The filter function returns a sequence on which a function is true.
